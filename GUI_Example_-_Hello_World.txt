This example will open an link:R3_GUI[R3 GUI] window with a title
message and a close button.


Script
~~~~~~

<pre> view [ title "Hello World" button "Close" close ] </pre>


Screenshot
~~~~~~~~~~

Note: uses development skin, not the final skin.

Image:hellorebol.png


Description
~~~~~~~~~~~

The TITLE face displays a title in a large bold font at the top of the
panel. It also sets the window title to the same string.

The BUTTON face when clicked with execute the CLOSE reactor, which will
close the window.

If no other windows are open, REBOL will quit.
