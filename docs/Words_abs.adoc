
ABS
^^^

Returns the absolute value. ABS is an action value.


Arguments
+++++++++

::
  *value* -- (Type: number! pair! money! time!)


Description
+++++++++++

Returns a positive value equal in magnitude.

 print absolute -123

print absolute -1:23
